# Λογική κώδικα micro:bit για ανίχνευση εμποδίων

Το αρχείο αυτό περιγράφει τη βασική λογική λειτουργίας του micro:bit στο έργο **Open Accessibility Hat**. Ο μικροελεγκτής λαμβάνει δεδομένα από αισθητήρα απόστασης και ενεργοποιεί ηχητική ή απτική ειδοποίηση όταν ανιχνεύεται εμπόδιο σε μικρή απόσταση.

## Βασική λογική λειτουργίας

1. Ο αισθητήρας απόστασης ελέγχει συνεχώς αν υπάρχει εμπόδιο μπροστά από τον χρήστη.
2. Η απόσταση μετατρέπεται σε αριθμητική τιμή σε εκατοστά.
3. Αν η απόσταση είναι μικρότερη από ένα προκαθορισμένο όριο, το micro:bit ενεργοποιεί προειδοποιητικό ήχο ή δόνηση.
4. Αν η απόσταση είναι μεγαλύτερη από το όριο ασφαλείας, η ειδοποίηση σταματά.
5. Η διαδικασία επαναλαμβάνεται συνεχώς σε πραγματικό χρόνο.

## Ενδεικτικός ψευδοκώδικας

```text
Για πάντα:
    μέτρησε την απόσταση από τον αισθητήρα

    αν η απόσταση είναι μικρότερη από 50 cm:
        ενεργοποίησε έντονη ειδοποίηση

    αλλιώς αν η απόσταση είναι μεταξύ 50 cm και 100 cm:
        ενεργοποίησε ήπια ειδοποίηση

    αλλιώς:
        σταμάτησε την ειδοποίηση
```

## Ενδεικτικός κώδικας τύπου micro:bit Python

```python
from microbit import *
import music

SAFE_DISTANCE = 100
DANGER_DISTANCE = 50

while True:
    # Η μεταβλητή distance αντιπροσωπεύει την απόσταση σε εκατοστά.
    # Στην τελική υλοποίηση η τιμή προέρχεται από τον αισθητήρα υπερήχων.
    distance = 0

    if distance < DANGER_DISTANCE:
        display.show(Image.NO)
        music.pitch(880, 200)

    elif distance < SAFE_DISTANCE:
        display.show(Image.WARNING)
        music.pitch(440, 200)

    else:
        display.show(Image.YES)
        music.stop()

    sleep(200)
```

## Σχόλιο

Ο παραπάνω κώδικας παρουσιάζει ενδεικτικά τη λογική λειτουργίας. Η τελική υλοποίηση προσαρμόζεται ανάλογα με τον τύπο αισθητήρα απόστασης, τις θύρες σύνδεσης και το περιβάλλον προγραμματισμού που χρησιμοποιείται, όπως MakeCode, Scratch ή Python για micro:bit.
